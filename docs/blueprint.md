# **App Name**: Elevate Folio

## Core Features:

- Dynamic Content Manager: Fetch all portfolio content, including profile, projects, experiences, achievements, skills, and certifications, directly from Google Sheets or CSV files.
- Hero & About Display: Showcase user profile with a full-screen background image, gradient overlay, and 'About Me' section, precisely matching the visual reference.
- Interactive Projects Showcase: Display projects in a horizontally scrolling container with auto-slide functionality (pausing on hover) and manual scroll capabilities, sourcing all details from the data sheet.
- Structured Experience Grid: Present experiences and roles in a responsive grid layout with dynamically numbered cards, fetching all descriptions and images from the spreadsheet.
- Categorized Achievements Tabs: Implement a tab-switching interface to filter and display achievements by category (e.g., 'Conference', 'Competition'), with all data sourced from the sheet.
- Animated Skills Marquee: Visually represent skills using an infinitely looping horizontal marquee animation of logos, with data extracted from the designated skills sheet.
- Certifications & Contact Information: Showcase certifications in a grid card format and provide clickable contact links (email, LinkedIn, WhatsApp, Instagram) dynamically populated from the spreadsheet.

## Style Guidelines:

- Primary interactive color: Muted, deep olive-green (#566039) for main text, buttons, and section titles, providing an elegant and earthy tone.
- Background color: Soft, creamy beige with a subtle green undertone (#E7E9E2) to create a bright, spacious, and calming canvas.
- Accent color: Desaturated golden-olive (#C2AD70) for subtle highlights or distinct interactive elements, maintaining the overall natural palette.
- Headline font: 'Literata' (transitional serif) for an elegant, literary, and refined feel, suitable for prominent titles.
- Body font: 'PT Sans' (humanist sans-serif) for clean readability and a touch of warmth in descriptive text, complementing the serif headlines.
- Consistent use of crisp, modern iconography, including actual brand logos for social media and skills, and country flag images for achievements, as depicted in the reference design.
- A spacious, clean, and exact layout following the provided design reference, featuring large negative space, full-screen hero sections, and mobile-first responsiveness for adaptability across devices.
- Smooth, fluid animations using Framer Motion, specifically for the auto-sliding project showcase (pausing on hover) and the infinitely looping marquee effect in the skills section.
