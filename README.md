# Signature Portfolio

This is a premium Next.js starter for high-end professional portfolios, built with precision and modern aesthetics.

## Deployment to Vercel

1. Push this code to your GitHub repository.
2. Connect your repository to [Vercel](https://vercel.com).
3. Configure spreadsheet env:
   - `GOOGLE_SHEET_ID` for direct Google Sheet (recommended), or
   - `SHEET_URL` for custom JSON endpoint.
4. Vercel will automatically detect Next.js and deploy your app.

## Spreadsheet-First Data Management

- Detailed setup ada di `docs/spreadsheet-maintenance.md`.
- Semua gambar disimpan di folder `public/assets`.
- Di spreadsheet cukup isi nama file gambar, contoh: `project-fintech.jpg`.

Built with Next.js 15, Tailwind CSS, and ShadCN UI.
